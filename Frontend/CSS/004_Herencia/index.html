Herencia

  • Control explicito
  • Ejemplo

Algunas propiedades CSS se heredan desde los elementos padres a los elementos hijos, modificando el valor que tienen por defecto

Flujo de la herencia

Si aplicas color: lightslategray; al body todos los elementos herendan el color.

Controlar explícitamente la herencia

Puede hacer que cualquier propiedad herede el valor calculado de su elemento primario con la palabra clave inherit. Una forma útil de usar esta palabra clave es crear excepciones.

Hereda font-weight: inherit;

Valor por defecto/Inicial font-weight: initial;

font-weight: unset; Si la propiedad se puede heredar es inherit else initial
