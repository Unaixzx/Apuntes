font-style

Permite definir el aspecto de una familia tipográfica

normal

italic

oblique

font-variant

La propiedad font-variant selecciona entre los aspectos normal y small-caps para la familia de fuente determinado.

small-caps:

Especifica un tipo de letra que está declarado como tipo de letra small-caps.

font-weight

Especifica el peso o grueso de la letra.

normal

bold

Relativo al padre:

lighter

bolder

100

900

font-stretch

Selecciona una cara normal, condensada o expandida de una fuente.

line-height

Establece la altura de una casilla remarcada por líneas. Comúnmente se usa para establecer la distancia entre líneas de texto

font-family

Define una lista de fuentes o familias de fuentes, con un orden de prioridad, para utilizar en un elemento seleccionado.
